假设有一个字符串，你想知道字符串中每一个字符出现的次数，可能有如下实现方式：
- 你可以创建26个变量，分别记录字母表中每个字符出现的次数。然后遍历字符串，每遇到一个字符，相应的计数变量就加1.

- 你可以创建一个包含26个元素的列表，然后使用Python内置ord函数将每一个字符转换成数字，把这个数字作为索引，再增加相应的计数元素。

- 你可以创建一个字典，用字符作为键，对应的计数作为值。当你第一次遇到一个字符时，添加一个键值对到字典中，之后你就可以增加字符对应的值了。

上面方法都实现了相同的计算结果，但是每种方法的计算策略各不相同。

执行计算的方式叫做实现，有一些实现要优于其它的实现，比如，字典实现的优势是我们最开始并不需要知道字符串中有包含哪些字符，我们只需要为那些真正出现的字符提供空间就可以了。

下面的可能的实现代码：
```python
word = 'brontosaurus' 
d = dict()
for c in word:
    if c not in d: 
        d[c] = 1
    else:
        d[c] = d[c] + 1
print(d)
```
for循环遍历整个字符串，每一个循环中，如果字符`c`不在字典里面，就创建一个新的键值对，将c作为键，初始化值为1，如果`c`已经存在于字典中，就增加`d[c]`。

下面是程序的输出：
```python
{'a': 1, 'b': 1, 'o': 2, 'n': 1, 's': 2, 'r': 2, 'u': 2, 't': 1}
```
字典有一个get方法，该方法以一个键和默认值作为参数，如果键存在于字典中，get方法就返回此键对应的值，否则，返回参数里面的默认值：
```python
>>> counts = { 'chuck' : 1 , 'annie' : 42, 'jan': 100}
>>> print(counts.get('jan', 0))
100
>>> print(counts.get('tim', 0))
0
```
我们可以使用get方法来更简洁地实现上面的计数程序。因为get方法自动处理了键不在字典的情况，可以将4行语句缩减至一条，还可以去掉if语句：
```python

word = 'brontosaurus' 
d = dict()
for c in word:
    d[c] = d.get(c,0) + 1
print(d)
```
使用get方法简化技术循环的做法已经成为Python中的一种普遍做法，在后续章节中会多次用到，所以你要仔细体会使用if语句和in操作符的循环和get方法之间的区别。它们做的事情是一样的，单某一个更高效。
