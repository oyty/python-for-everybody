**习题 2.3**
编写一个程序，输入工时和时薪，然后计算出工资总额。
```
Enter Hours: 35
Enter Rate: 2.75
Pay: 96.25
```
暂时不用担心计算结果是否精确到小数点后两位。如果要精确到小数点后两位，可以使用Python内置的round函数。
```
round(1.95583, 2)  # 1.96 
```
