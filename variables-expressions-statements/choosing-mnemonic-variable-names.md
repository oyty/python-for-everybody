只要你遵循变量命名的简单规则，避免使用关键字，变量的命名还是有很多选择的。在初入门阶段，你在阅读别人的程序或者编写自己的程序时可能会感到困惑。例如，下面三个程序实质上是一样的，但是阅读和理解起来差别很大。

```python
a = 35.0 
b = 12.50 
c=a*b 
print(c)
```

```python
hours = 35.0
rate = 12.50
pay = hours * rate
print(pay)
```

```python
x1q3z9ahd = 35.0
x1q3z9afd = 12.50
x1q3p9afd = x1q3z9ahd * x1q3z9afd
print(x1q3p9afd)
```
