取余运算对整数进行操作，在Python中取余运算符是`%`，语法和其它操作符一样：
```python
>>> quotient = 7 // 3
>>> print(quotient)
2
>>> remainder = 7 % 3
>>> print(remainder)
1
```
所以`7`除以`3`等于`2`余`1`。

取余运算符非常有用，比如你可以用来判断一个数是否能被另一个数整除，如果`x % y`为`0`，则`x`能被`y`整除。

你也可以使用取余运算从一个数字中取出后面的数字，比如`x % 10`得到的是个位上的数，相似的，`x % 100`得到的就是最后的两位数。
